[getbem.com](http://getbem.com)
=====================

This repository contains the source for the [getbem.com](http://getbem.com) site.

### Usage

```
npm install
npm start

open http://localhost:3000
```

### Linting

This project includes React-friendly ESLint configuration.

```
npm run lint
```

### Deploying

```
npm run publish
```
